# Geometry_Game

A simple game to guess point coordinates and calculate the square of a rectangle.

## Introduction

Welcome to Geometry_Game! This is a simple Python game designed to help you practice your geometry skills. The game challenges you to guess the coordinates of points and calculate the square of rectangles. It's an engaging way to reinforce your understanding of basic geometric concepts.

The primary workflow involves running the game from the command line, entering coordinates or dimensions as prompted, and receiving feedback on whether your answers are correct.

## Features

- **Guess Point Coordinates**: Practice identifying the x and y coordinates of points.
- **Calculate Rectangle Square**: Test your ability to calculate the area of rectangles given their dimensions.

## How It Works

The game is built using Python. The primary entry point for running the application is `main.py`. When you execute this script, it will prompt you to enter the necessary information and provide feedback on your answers.

2. Navigate to the project directory:
